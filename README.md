
# 2025 F1 Rankings – Static Site

A minimal static website ready for GitHub Pages. Update `data/standings.json` as the 2025 season evolves, commit, and Pages will auto-deploy.

## Quick edit
- Edit `data/standings.json` directly on GitHub → Commit → wait ~1 min → refresh.

## Local preview (optional)
Open `index.html` directly in your browser. No build step needed.
